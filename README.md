# 実行環境
  OS ubutu 20.04
  ROS1
  gTTS 
  
# 環境構築

    git clone
    cd ~/catkin_ws
    catkin_make
    source ~/.bashrc
    
    pip3 install gtts
    
#実行方法
Google Translate's text-to-text speech APIを使用するためインターネットにつながった状態で実行してください。

1. roscoreを立ち上げてください

    roscore
    
2. gTTSをインストールします

    pip3 install gtts
